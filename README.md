# Diventa

Sito web di Diventa — L'Evoluzione Digitale e Umana della tua Organizzazione.

## Struttura

- `index.html` — Homepage
- `contatti.html` — Pagina contatti
- `css/style.css` — Stili
- `js/main.js` — Menu mobile e interazioni
- `img/` — Immagini e loghi

## Come usarlo

Aprire `index.html` nel browser. Non servono build tool o dipendenze, è un sito statico in HTML/CSS/JS puro.
